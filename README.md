# TechStore E-commerce

## Project Overview
TechStore is a modern e-commerce application designed to provide users with a seamless shopping experience. Users can browse a variety of products, manage their shopping cart, and complete their purchases with ease.

### Features
- User Authentication
- Product Browsing
- Shopping Cart Management
- Order History

## Setup Instructions
To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/trantungzm/TechStore-Ecommerce.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd TechStore-Ecommerce
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the application**:
   ```bash
   npm start
   ```

5. **Open your browser** and go to `http://localhost:3000` to view the app.
